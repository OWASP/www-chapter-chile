# Documentos OWASP Chile

Esta carpeta contiene documentos y recursos del capítulo OWASP Chile.

## Sobre OWASP Chile

OWASP Chile está comprometido con fortalecer la seguridad de aplicaciones en nuestra región a través de:

- **Educación gratuita y accesible** para toda la comunidad de desarrollo y seguridad
- **Eventos regulares** donde compartimos conocimiento y experiencias
- **Colaboración abierta** con profesionales, empresas y estudiantes
- **Contenido en español** adaptado a nuestra realidad local
- **Promoción de estándares internacionales** de seguridad de aplicaciones

## Nuestra Misión

Apoyar la creación de software más seguro en Chile mediante la difusión de conocimiento, herramientas y mejores prácticas de seguridad aplicativa, promoviendo una cultura de desarrollo seguro en organizaciones públicas y privadas.
